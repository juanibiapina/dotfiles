Increase visual impact and personality in designs that are too safe, generic, or visually underwhelming. Use the frontend-design skill for design principles and anti-patterns — bold means distinctive, not "more AI effects."

$ARGUMENTS

## Understand Context First

Gather target audience, use cases, and brand personality from the conversation or codebase. If unclear, ask before proceeding. "Bolder" without context produces generic AI slop.

## Assess

Identify what makes the design feel safe: generic font choices, timid scale (everything medium-sized), low contrast between elements, static/no motion, predictable layouts, flat hierarchy where nothing stands out.

## Amplification Dimensions

### Typography
Replace generic fonts with distinctive choices. Create dramatic size jumps (3x-5x, not 1.5x). Pair extreme weights (900 with 200). Use variable fonts, display fonts, or condensed/extended widths.

### Color
Increase saturation with intentional, non-AI-slop palettes. Let one bold color own 60% of the design. Use sharp accent colors and tinted neutrals. Rich multi-stop gradients, not generic purple-to-blue.

### Spatial Drama
Extreme scale jumps for important elements. Break the grid — let hero elements escape containers. Asymmetric layouts with tension. Generous white space (100-200px gaps). Intentional overlap for depth.

### Visual Effects
Dramatic shadows for elevation. Background treatments (mesh, noise, geometric patterns). Texture and depth (grain, halftone, duotone). Custom decorative elements that reinforce brand.

### Motion
Staggered entrance choreography. Scroll-triggered reveals. Satisfying hover effects and click feedback. Smooth transitions with ease-out-quart/quint/expo easing.

### Composition
Clear focal points with dramatic treatment. Diagonal flows, full-bleed elements, unexpected proportions (70/30, 80/20 splits).

## Constraints

- Bold design must still be usable — impact without function is decoration
- If everything is bold, nothing is — need contrast between loud and quiet elements
- Body text must remain readable
- Must still meet WCAG accessibility standards
- Don't add effects randomly without purpose
- Don't copy trendy aesthetics blindly — bold means distinctive, not derivative
- If the result looks like typical AI "bold" output (neon on dark, glassmorphism, purple gradients), start over

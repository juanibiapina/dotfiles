# Manually Installed Software

Software installed manually (not via Nix or Homebrew).

## slackcli

- **Repo:** https://github.com/shaharia-lab/slackcli
- **Location:** `~/bin/slackcli`
- **Install:**
  ```bash
  curl -L https://github.com/shaharia-lab/slackcli/releases/latest/download/slackcli-macos-arm64 -o slackcli
  chmod +x slackcli
  mv slackcli ~/bin/
  ```
